# 📅 Cronograma y RoadMap de Entregas

| Hito | Título | Semanas | Enfoque Principal |
|------|--------|---------|-------------------|
| **Hito 1** | Diseño y Modelado | 1 — 5 | Modelado de dominio (UML), estados y diseño de contratos API |
| **Hito 2** | Backend y Lógica | 6 — 10 | Implementación en Spring Boot, persistencia con ORM y motor de reglas |
| **Hito 3** | Frontend y Seguridad | 11 — 15 | Interfaz en Angular, consumo de API, seguridad JWT y despliegue |

---

## Hito 1 — Diseño y Modelado (Semanas 1 a 5)

- Diagrama de clases UML del dominio
- Diagrama de estados del ciclo de vida de la solicitud
- Diseño de contratos de la API REST (endpoints, DTOs, códigos de respuesta)
- Estructura de carpetas del proyecto definida
- Repositorio GitHub configurado

## Hito 2 — Backend y Lógica (Semanas 6 a 10)

- Proyecto Spring Boot funcionando
- Entidades JPA mapeadas y base de datos configurada
- Todos los RF del 01 al 08 implementados en el backend
- API REST documentada con Swagger/OpenAPI
- Seguridad JWT configurada (RF-13)
- Tests unitarios de la capa de servicio

## Hito 3 — Frontend y Seguridad (Semanas 11 a 15)

- Proyecto Angular conectado al backend
- Formularios de registro, login y gestión de solicitudes
- Tablas con filtros y paginación
- Guards y roles implementados en el frontend
- Integración opcional con IA (RF-09, RF-10, RF-11)
- Despliegue en plataforma cloud
